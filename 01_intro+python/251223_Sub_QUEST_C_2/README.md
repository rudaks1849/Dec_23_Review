# PRT(Peer Review Template)
- [x]  **1. 주어진 문제를 해결하는 완성된 코드가 제출되었나요?**
    - 네 대단히 훌륭한 작업물이 제출되었습니다.
    - *코드 길이가 긴 관계로 사진은 생략합니다*

<br><br>

    
- [x]  **2. 전체 코드에서 가장 핵심적이거나 가장 복잡하고 이해하기 어려운 부분에 작성된 
주석 또는 doc string을 보고 해당 코드가 잘 이해되었나요?**
    - 모든 프로세스마다 설명 주석을 달아주신 부분이 가장 인상적인 부분이었습니다.
    - 왜 이 함수와 명령어를 사용했는지 친절하게 설명해주셔서 학습에 도움이 되었습니다.

- [x]  **3. 에러가 난 부분을 디버깅하여 문제를 해결한 기록을 남겼거나
새로운 시도 또는 추가 실험을 수행해봤나요?**
    - 가장 하단에 [추가수행] 이라는 부분을 작성해주셔서, 반복되는 프로세스를 간단히 함수화하여 전체 코드를 간소화하는 작업을 시도하셨습니다. 저희는 이런 부분을 고려하지 못했는데, 도움이 많이 되었습니다.

- [x]  **4. 회고를 잘 작성했나요?**
    - 전체적으로 주석을 통한 설명이 매우 디테일해서 어떤 의도로 각 작업을 진행했는지 한 눈에 알기 쉬웠습니다.

<br><br>

- [x]  **5. 코드가 간결하고 효율적인가요?**
    - 네. 매우 깔끔하고 효율적으로 작성되었습니다. [추가 수행] 부분 뿐만 아니라, 대부분의 코드들에 군더더기 없어 술술 읽혔습니다.
